# Calculadora em Java

Este é um projeto simples de uma **calculadora em Java** que realiza operações básicas: adição, subtração, multiplicação e divisão.

## Funcionalidades

- Adição, subtração, multiplicação e divisão
- Tratamento de divisão por zero
- Validação de operações inválidas
- Exibição do resultado no console
